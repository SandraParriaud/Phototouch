# Phototouch
Phototouch is a touch based photobooth. It will enable you to take two picture of two people and to mix it so that you obtain a new picture : a meeting picture.


### Required materials
1 Arduino uno
1 Ultrasonic sensor 
3 Jumper cables

1 Makey makey
2 Alligator Clips

2 screens
